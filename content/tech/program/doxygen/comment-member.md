メンバ変数の後ろにドキュメント・ブロックを置くことができます。

```
class Hoge {
    ...

    int var1;  ///< ドキュメント
    int var2;  //!< ドキュメント
    int var3;  /*!< ドキュメント */
    int var4;  /**< ドキュメント */
};
```

`enum` の各エントリに対しても同様にドキュメントを記述することができます。
このように、行末に1行で記述したドキュメントは、Brief description として扱われます。
