`du` コマンドを使って、ディレクトリ内のファイルの合計サイズを調べることができます。

{{< code >}}
$ du -hs mydir
685M    mydir
{{< /code >}}

各オプションは次のような効果があります。

- `-h (--human-readable)` ... 人に読みやすい単位でサイズ表示します（例: 1K、234M、2G）
- `-s (--summarize)` ... 合計サイズのみを表示します

サイズ部分だけを表示したい場合は、`cut` コマンドで 1 番目のフィールドを切り出してしまうのが手っ取り早いです。

{{< code >}}
$ du -hs mydir | cut -f1
685M
{{< /code >}}
