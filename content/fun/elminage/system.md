種族（全9種類）
----

- <b>人間</b>
  - バランスの取れた能力値。
    だから逆に選ばれにくい？
- <b>エルフ</b>
  - 呪文が得意です。
    エルブンチェーン+1 を装備することができるのはエルフだけです。
- <b>ドワーフ</b>
  - 力と信仰を備えた種族です。僧侶に向いています。
- <b>ノーム</b>
  - 神に従順で信仰心が厚く、僧侶に向いています。
- <b>ホートルット（ホビット)</b>
  - 職業「盗賊」に向いています。
- <b>フェアリー</b>
  - **敏捷性** が高いですが、**装備品に制約** が多いです。
- <b>ワービースト</b>
  - 通常攻撃で毒爪によって毒を与えられます。
    職業「狩人」はターン終了時に状態異常の敵に追撃できる技能を持っているため、**ワービースト狩人** がおすすめです。
    **銀製品** を装備すると呪われてしまいます。
- <b>ドラゴニュート</b>
  - 使用回数無制限のブレスは敵1列を攻撃でき、中盤までの主力攻撃になります。
  - ブレスは **HP** が高いほど威力が上がるので、**ドラゴニュート戦士** がおすすめです。
  - ブレスは最初は単体攻撃ですが、Lv.8 から前列全体を攻撃できるようになります。
  - 敏捷が低いため、魔術師などには向いていません。
  - ドラゴンキラーを装備すると呪われてしまいます。
- <b>デビリッシュ</b>
  - 悪魔と人間のハーフで、**呪われた武具** を装備できます。
  - 職業「闘士」用の武器は呪われているものが多いので、**デビリッシュ闘士** がおすすめです。
- <b>魔傀儡 ※</b>
  - 呪文によって命を宿した人形です。
  - 呪文抵抗は高い (+35%) ですが、物理攻撃を避けにくい (基本AC:15）です。
  - 状態異常回復率 (15%) は、が他の種族より 15% 低いです。
- <b>オーガ ※</b>
  - 腕力が高く戦士に向いています。
- <b>ゴブリン ※</b>
  - 力で戦うよりも頭脳戦を得意とします。


※イベントクリア後に作成可能。


職業（全16種類）
----

宝箱の罠をはずしたり、鍵のかかった扉を開けるために「盗賊技能」を持つキャラはほぼ必須。
司教の「鑑定」があると、ダンジョン内でもアイテムの鑑定ができて便利。

### 基本職（6種類）

- <b>戦士</b>
  - 特徴: 多くの武器・防具を装備可能。呪文は不可。最初から購入できるフランベルジュ (15000GP) は序盤で活躍する。
  - 基本技能: なし
  - ハイマスター能力: **物理攻撃力UP**
- <b>魔術師</b>
  - 特徴: 魔術師呪文を習得。序盤の攻撃呪文が心強い。
  - 基本技能: **魔術師呪文**
  - ハイマスター能力: **呪文威力UP+1**
- <b>僧侶</b>
  - 特徴: 僧侶呪文を習得。貴重な回復役。
    霊体攻撃の技能を持っており、霊・不死・悪魔系のモンスターに対して物理攻撃でダメージを与えることができます。
  - 基本技能: **僧侶呪文**／ディスペル／霊体攻撃
  - ハイマスター能力:
- <b>盗賊</b>
  - 特徴: 罠解除や隠し扉の発見が可能。**パーティにほぼ必須**。
    狩人と忍者も盗賊技能は持っていますが、盗賊ほど信頼できません。
  - 基本技能: **盗賊技能+1**
  - ハイマスター能力: **盗む**（装備解除した敵から装備を盗むことができる）
- <b>錬金術師</b>
  - 特徴: 錬金呪文を習得。錬金技能で錬金アイテムを作成可能。鍛治技能で装備品を強化できます。
    レベルが上がると、鉱石の練度にボーナスを付けて錬金・鍛治を行うことができます。
    ハイマスター能力の分解が使えるようにあると、強化した装備品を元に戻して鉱石を分離できます。
  - 基本技能: 錬金呪文／**鍛冶**／**錬金**
  - ハイマスター能力: **分解**（鍛冶で強化した武具を元に戻す。鉱石を分離できるので、気軽に別の武器を強化できる）
- <b>使用人</b>
  - 特徴: ターン終了時に自動でアイテム使用。使用人鞄に回復アイテムを99個ずつ詰め込める。
  - 基本技能: **メイドリカバー**／貯金箱／ティータイム
  - ハイマスター能力:
    - 迷宮のお茶会: 「紅茶葉」の MP 回復量が 1 から 2 に増加。

基本職は Lv26 でハイマスター能力が解放。

### 中級職（7種類）

- <b>司教</b>
  - 特徴: 魔術師呪文と僧侶呪文を習得できますが、覚えるのは遅いです。
      その場で鑑定が可能なので、冒険中に見つけた不要なアイテムをその場で捨てることができます。
  - 基本技能: 魔術師呪文／僧侶呪文／ディスペル／**鑑定**
- <b>狩人</b>
  - 特徴: 弓で先制攻撃が可能。ターン終了時に状態異常の敵に追撃できる技能を持っています。
  - 基本技能:
- <b>闘士</b>
  - 特徴: 拳で攻撃。連続攻撃や耐性を消すクリーンヒットが発生。
- <b>遊楽者</b>
  - 特徴: 呪文・盗賊技能・演奏・タロットを扱う。
  - 基本技能: 魔術師呪文／盗賊技能／タロット／楽器演奏
- <b>巫</b>
  - 特徴: 霊・不死・悪魔に強い。結界を展開可能。
  - 基本技能: ディスペル／霊体攻撃／楽器演奏／結界
- <b>召喚士</b>
  - 特徴: モンスターを召喚可能。
  - 基本技能: 召喚呪文
- <b>神女</b>
  - 特徴: 槍で後列攻撃。僧侶呪文を習得。**中立女性のみ**。
  - 基本技能:

中級職は Lv32 でハイマスター能力が解放。

### 上級職（3種類）

- <b>君主</b>
  - 特徴: 特殊攻撃に耐性。
  - 基本技能:
- <b>侍</b>
  - 特徴: メイン武器を二刀流可能。魔術師呪文を習得。
  - 基本技能: 魔術師呪文／メイン武器二刀流
- <b>忍者</b>
  - 特徴: 一撃必殺の首切り。
  - 基本技能:

上級職は Lv36 とハイマスター能力が解放。

### 職業能力

- **基本能力**: 各職業には最初から備わっている能力。
- **ハイマスター能力（サマー能力）**: 特定レベルに達することで習得できる能力。
- 例:
  - 司教「鑑定」 ... 未鑑定アイテムを鑑定することができる。酒場かダンジョンの「まとめて鑑定」メニューから鑑定できる。呪われたアイテムだと「さわってしまった！」となって鑑定できないことがある（しばらく歩いているとまた鑑定できるようになる）。
  - 盗賊「装備解除」 ... モンスターの装備を外して戦闘力を低下させる。「盗む」ために先にこれで装備を外す必要がある。
  - 盗賊「盗む」 ... 「装備解除」で外したモンスターの装備を盗むことができる。
  - 遊楽者「タロット」 ... タロットカードを引いて独自の効果を得る。ドローに失敗（逆位置）すると対象相手が逆になる。
  - 使用人「メイドリカバー」 ... 「使用人鞄」に回復アイテムを入れておくことで、ターン終了時に自動的にアイテムを使用してくれる。


能力値 
----

- レベルアップすると 6 つの能力値が増減する。年齢が若いと上がりやすく、高齢だと下がりやすい。
- 能力値の最大値: 各種族の基本値に **+10** したもの。

| 能力値 | 説明 |
| ---- | ---- |
| 腕力 | 物理攻撃の当てやすさ（ダメージが上がるわけではない） |
| 知恵 | 魔術師呪文と錬金術呪文の覚えやすさ |
| 信仰 | 僧侶呪文の覚えやすさ |
| 体力 | 最大 HP の増えやすさ |
| 敏捷 | 戦闘時の攻撃順、罠の判別や解除といった盗賊技能に影響 |
| 幸運 | 盗賊技能に関する行動に影響 |


名前／ニックネーム
----

- 「名前」と「ニックネーム」は両方つけることが必須。
- あとから自由に変更できるので、適当につけておいてもよい。


呪文抵抗
----

- 「魔術」「僧侶」「錬金術」「召喚」の 4 種類の呪文抵抗（0〜100%）がある。
- 抵抗に成功すると、呪文の効果を完全に無効化できる。


属性攻撃／属性防御
----

- 属性攻撃／属性防御は、主に装備品により付加できる。
- 属性攻撃がついていると、攻撃時の属性ダメージ（火、氷、雷、神聖、暗黒）が増加する。
- 属性防御がついていると、敵からの属性攻撃を軽減できる。
  属性防御が100％以上の場合は無効化、200％以上は吸収（ダメージ分回復）、300％以上になると反射になる。


倍打
----

- 倍打は、主に装備品により付加できる。
- 物理攻撃を相手に与えた際、その種族に対する倍打効果があるとダメージ量が変化する。
- △:0.5倍、－:1倍、=:1.5倍、○:2倍、☆:3倍、S:5倍、!:10倍


付加効果発動／付加効果抵抗
----

- 主に装備品により付加できる。
- 付加効果発動: 物理攻撃を行った際、相手に「睡眠」「猛毒」といった状態異常や、「首切り」のような付加効果を発動させる確率。
- 付加効果抵抗: 付加攻撃を受けた際に抵抗する確率を表します。抵抗に成功すると、その効果は無効になる。


状態回復率／ターン回復
----

- 主に装備品により付加できる。
- 状態回復率: ターン終了時や移動時に「睡眠」「沈黙」「混乱」「魅了」「恐怖」といった状態異常を回復させる確率。
- ターン回復: ターン終了時の HP の回復値。

年齢
----

- **年齢が低いほどレベルアップ時の能力値が向上しやすい** ので、基本的には低い方がよい。
- ただし、年齢が高いほど攻撃力が上がる武器もあるらしい。
- 年齢が低いとキャラ作成時のボーナスを得られにくいが、高いボーナスが出るまで粘ればよい。
- 宝箱の罠「**女神の口づけ**」にかかると、年齢を下げることができるので、あまりこだわり過ぎなくてよい。
  - 光の庭3Fで確認。


性別／性格
----

- 性別は「男」「女」の 2 種類。
- 性格は「善」「悪」「中立」の 3 種類。
- 性別と性格により、就ける職業が決まる。
- 「善」と「悪」のキャラクターは同じパーティに入れることができない。
  あるイベントをクリアすると「善」と「悪」でパーティを組めるようになるらしい。


AC（アーマークラス）
----

- AC が低いほど敵からの物理攻撃を回避しやすくなる。具体的にはヒット回数が減ることでダメージが減る。
- 呪文に対しては効果はない。


攻撃回数
----

- 武器ごとに攻撃回数が決まっていますが、キャラクターのレベルによっても追加されます。
  - 5 レベル上がるごとに +1 回される職業（8種）: 戦士/狩人/闘士/巫/神女/君主/侍/忍者
  - 10 レベル上がるごとに +1 回される職業（4種）: 僧侶/盗賊/使用人/遊楽者
  - 攻撃回数が上がらない職業（4種）: 魔術師/錬金術師/司教/召喚師
- 攻撃回数の最大値は 10 回。


転職
----

- 転職の条件は、転職先の能力値を満たしていること（あとは性格と性別も）。
- 転職後に引き継がれるのは **HP** と **呪文** のみ。経験値も 0 に戻ります。
- 能力値は転職先の基本値になります（ボーナス値など一切なし）。
- MP は同系列の呪文を覚える職業でない限り、全レベル 3 になります（各レベルの呪文が 3 回まで使える）。
- 本タイトルではそれほど転職のメリットがありません。


錬金倉庫
----

- 「ミアマックの宿屋」で 300GP を支払うことで「錬金倉庫」を利用できる。
  「倉庫」は誰でも使えるけど、「鍛冶」「錬金」「分解」は錬金技能を持つキャラクターがいないと使用できない。
- 入るたびに 300GP かかるのは痛いけど、イベント「ミアマックの心配事」をクリアすると無料になるらしい。
- 錬金倉庫でできること
  - **鍛冶** ... 装備品と鉱石（素材）を合成して、様々な付加効果を持つ装備品を作り出す。
    鉱石は練度という数値を持っていて（例:30）、これを付加するSKILLに割り当てることで装備品を強化する。
    各強化にはAPという練度消費量が決められていて、鉱石の練度の範囲で振り分けることができる。
    例えば、「射程制限解除(AP:30)」「呪い解除(AP:10)」などに割り当てる。
    1つの鉱石しか合成できないので、他の鉱石を合成したいときは一度「分解」で装備品を元の状態に戻す必要がある。
  - **錬金** ... 鉱石と鉱石を合成して、錬金アイテムと呼ばれる特殊アイテムを作り出す。
  - **分解** ... 「鍛冶」で合成した装備品と鉱石を元の状態に戻す。
  - **倉庫** ... 所持しているアイテムを倉庫に保管する。

{{% accordion title="イベント「ミアマックの心配事」" %}}
1. 最初の洞窟のボスを倒して、王様に 5 つの指輪探しの依頼を受けるところまで進める。
1. 宿屋の主人と 3 回話して「へーメの頭痛薬」の話題を聞くと、Event No.10「ミアマックの心配事」が発生する。
1. ヘーメ共同地区のニャンニャンどぐぁに行き、頭痛薬 (5000GP) を買い、宿屋の主人に渡す。

フェイム
----

- コンパスにフェイムを3個以上献上すると、目的とする物の在り処を示してくれる。
  たくさん献上すると反応しやすくなる（最大7個）。
- ダンジョン内で「コンパス」→「サーチ設定」で、目的とするものを選択しておくと、画面上に黄色いラインが出てくる。

| フェイムの数 | コンパスの反応 |
| --- | --- |
| 3個 | 周囲 1 マスに指輪があると反応 |
| 4個 | 周囲 4 マスに指輪があると反応 |
| 5個 | 現在の階層に指輪があると反応 |
| 6個 | 現在のダンジョンに指輪があると反応 |
| 7個 | 他のダンジョンに指輪があっても反応 |
