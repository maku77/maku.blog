Nintendo Switch の SEGA AGES で初代『**ファンタシースター**』が売ってたので懐かしすぎて買ってみました。
メガドライブよりも前の、**セガ・マークIII／マスターシステム** というゲーム機で発売されたタイトルです（メガドライブで復刻版も出たけど）。
幼い頃に兄がやっていたのを横で見ていた記憶があります。
メガドライブにアダプターを指してマークIIIのカートリッジを差して遊んでいたのを覚えています。
3D ダンジョンや敵がぬるぬる動いているのがすごいなぁと思っていました。

この Switch バージョンには、さくさく進める **AGES モード** というのついていて、敵を倒した時に手に入るメセタ（お金）や経験値が増えるようになっています。
AGES モードというのは「おじさん世代モード」ってことでしょうか。
サクサク進めたいのはどちらかというと Z 世代な気もするけど。

ストーリーはとてもシンプルで、兄を殺されてしまったアリサが、兄の仇を討つために諸悪の根源ラシークを倒そうというもの。
動機は世界を救うとかではなくて、あくまで **個人的な復讐** です。
シンプルでいいですね（いいのか）。

{{< image src="img-myau.jpg" title="かわいいミャウ（いきなり喋る猫）" >}}

最初の仲間のミャウとタイロンはすぐに仲間になります。
猫のミャウは「ビンの蓋をあけられない」というけれど、魔法や武器での攻撃をバリバリこなし、人間の言葉まで話せるあんたなら **ビンの蓋くらい開けられるでしょう**、とはツッコミますまい。
あくまでかわいい猫です。

タイロンはメデューサに石にされてしまったらしく、それをアリサとミャウが助けるのですが、石にされた理由というのがこちら。

{{< image src="img-tylon.jpg" title="アリサ「なぜ無謀にもメデューサに立ち向かったの？」" >}}
{{< image src="img-tylon2.jpg" title="タイロン「斧が欲しくって...」" >}}

しょうもない...。
しかも「逃げられちまった」って、あんた石にされてるじゃん。

ちなみに Switch 版のいいところは、`+ (Start)` ボタンを押せばいつでも魔法やアイテムの説明が見られるところです。
おかげで、魔法の名前を最後まで覚えられませんでした（ダメじゃん）。

{{< image src="img-magic-list.jpg" title="ポーズメニューが超便利" >}}

回復魔法「ヒール」の上位版の名前が「ドヒール」っていうのも **昭和の香り** がしていいですね。
回復魔法はこの 2 種類しかありません。
このシンプルさがよいのです。
最近のドラクエとか、魔法や特技がインフレしすぎてて全然使わないものが多すぎるので、1画面に収まるくらいがちょうどいいんです。
それでも魔法の名前覚えられなかったけど。

あと、オートマッピングはやっぱり便利ですね。
2、3 日でクリアできたのはこれがあったからだと思います。
昔は手描きで全部マッピングしてたので、現代人からすると恐ろしいほど時間がかかっていたはず。
ある意味、娯楽の種類が限られていて、**マッピングに時間をかけるのが楽しかった時代** は幸せだったのかもしれません。

謎解きは比較的簡単だったけど、最後にエアロキャッスルに行くためのアイテム集めは少しだけ難解でした。
街の人のヒントを聞いても結局よくわからずに、アイテム総当たり試したり↓

{{< youtube id="WtNu9JbyisY" >}}

空に浮かんだ城を見て、一瞬でイースを思い浮かべた人は昭和ゲーマーです。

{{< image src="img-myau2.jpg" title="猫こそ最強（猫なのかは知らない）" >}}

無事、個人的な復讐劇を果たしたアリサは得意げの記念撮影です。
ミャウさんいつ小さな姿に戻ったの。

{{< image src="img-cheese.jpg" title="はい、チーズ！（昭和風に）" >}}

当時この絵を入れるために、プログラム容量をがんばって削ったのだとか。

スタッフロールがダンジョン探索風になっているのもオシャレ。
アイコンはアレックスキッドかな。

{{< image src="img-staff-roll.jpg" title="エイプリル・フールって..." >}}

それにしても、タイロンさんは MP が 0 なのに、HP まで最下位なのはかわいそすぎでした。
配置も後衛だし、そういうことなんでしょう。
猫が正義ということで。
チャンチャン（昭和風に）。
