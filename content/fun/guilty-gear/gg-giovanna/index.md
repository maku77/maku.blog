Guilty Gear Strive のジオヴァーナ練習メモです。
だんだん長くなってきたので [GGST基礎練メニュー](/p/73d9ami/) から分離しました。
ジオは他のキャラと比べてずるさを感じにくいのでよいです。
逆にいうと簡単にガードを崩せるような連携がないので、いろいろ工夫して動く必要があります。

コマンド
----

- セパルトゥア {{< gg "214K(セパル)" >}}。連携／コンボの締めに使う。ガードされたあとは確反にはならないが、相手ターンになってしまうので基本ガードで耐える。
- トロヴァォン {{< gg "236K(トロヴァ)" >}}。突進技。ガードさせれば有利なので固め継続できる。慣れている人には {{< gg "6P" >}} で簡単に落とされる。
- ソウ・ナセンテ {{< gg "623S(ナセンテ)" >}}。無敵時間のない対空技。ステップから出すのも有効。
  {{< gg "623S(近距離ナセンテ) BRC" >}} のガード崩しも強力。
  ガード時はジオ側 -4F なので、ガード、バクステ、{{< gg "2K" >}}、{{< gg "6P" >}}、{{< gg "623S(ナセンテ)" >}} などでフォロー。
- ソウ・ポエンチ {{< gg "236S(ポエンチ)" >}}。飛び上がり攻撃。
  {{< gg "2HS" >}} からのコンボに組み込むのが基本。
  空中では前後に移動可能。
  ガードされた場合は {{< gg "(空中)RRC j.P(x2)" >}} でガード崩し。
  {{< gg "めくりj.K > 236S(空中ポエンチ)" >}} というコンボもあり。
- シャーヴィ: {{< gg "214HS(シャーヴィ)" >}}。
  - 派生 {{< gg "6K" >}} トロヴァ。ガードさせて有利。ヒットしたらコンボへ。無敵技のない相手には起き上がりにうまく重ねるとよい。
  - 派生 {{< gg "6S" >}} ナセンテ。相手が浮いている場合、ここからコンボの起点に（開幕 {{< gg "HS counter-hit" >}} 時など）。
  - 派生 {{< gg "4K" >}} セパル。コンボの締めに。中距離から急に出す手もあり。
  - 派生 {{< gg "4S" >}} ポエンチ。めくりに使える。{{< gg "dash HS > 214HS(シャーヴィ) 4S(ポエンチ)" >}} でめくりからのコンボも可能。
- ヴェンターニア（覚醒）{{< gg "632146HS(ヴェンタ)" >}}。唯一のリバサ無敵技。ガードされたら終了。
- テンペスターヂ（覚醒）{{< gg "236236HS(テンペ)" >}}。2.00 以降はガードされると不利になったため使い所が減った。


立ち回り／ガード崩し／セットプレイ
----

### 起き攻め／近距離

ガードされたときに {{< gg "214K(セパル)" >}} まで繋げても全部ガードされるだけなので、いかに「投げ」や「ディレイ」、「最速ダストロマキャン」、「低空ロマキャン」、「めくりポエンチ」で揺さぶっていくかがポイント。

- {{< gg "c.S dash-cancel 投げ" >}}。ガード崩し。
- {{< gg "c.S dash-cancel c.S" >}}。固め継続。
- {{< gg "c.S delay 2S HS" >}} バクステ狩り＆暴れ潰し。ヒット時は {{< gg "214K(セパル)" >}}、ガード時は {{< gg "236K(トロヴァ)" >}} 固め。
- {{< gg "c.S delay f.S(x3) HS" >}} 暴れ潰し。
- {{< gg "c.S f.S(x3) delay HS" >}} 暴れ潰し。
- {{< gg "c.S f.S(x3) HS > 214HS(シャーヴィ) ~ 4S(ポエンチ) [6] 2K 6HS(x3)" >}}。連携をガードされたときのポエンチめくり崩し。ポエンチが当たったら、{{< gg "2K" >}} で拾える。
- {{< gg "c.S 5D 66RCC" >}}。近Sからの最速ダスト中段（直でもOK）。テンションゲージ溜まっているとき限定。
- {{< gg "c.S jamp-cancel 66BRC j.HS (land) j.HS" >}}。低空青ロマ中段 → F式。
- {{< gg "c.S jamp-cancel j.HS 22RRC land 2S HS" >}}。下スライド赤ロマ下段。
- {{< gg "214S [6](めくりポエンチ)" >}} / {{< gg "214HS(シャーヴィ) ~ 4S [6](めくりポエンチ)" >}}。表裏。いずれも {{< gg "6" >}} 入れっぱで表になる。シャーヴィからの方がダメージ大。
- {{< gg "j.HS air-dash j.HS" >}}。ガード崩し。
- {{< gg "c.S f.S(x3) HS > 236K(トロヴァ) 2K 2D 236K(トロヴァ) ..." >}} なめぷ永久トロヴァ固め。{{< gg "6P" >}} に負ける。画面端でトロヴァがヒットしたら追撃可能。
- {{< gg "シミー → 空投げ" >}} ファジージャンプ返し＆リバサ昇竜すかし。相手が飛ぶかだけに集中。

### 中距離

- 地上ダッシュからの連携
  - {{< gg "d.f.S(x3) HS > 214K(セパル)" >}}。主戦力。
  - {{< gg "d.2S HS > 214K(セパル)" >}}。主戦力。
  - {{< gg "d.HS > 214K(セパル)" >}}。{{< gg "d.HS(ch) > 214HS(シャーヴィ) ~ 6S(ナセンテ) 2K 6HS(x3)" >}}。主戦力。
  - {{< gg "d.K 6P > 214K(セパル)" >}}。主戦力。
  - {{< gg "d.K 2D > 214K(セパル)" >}}。主戦力。
  - {{< gg "d.K 2K 2D > 214K(セパル)" >}}。
    ちょっと特殊な {{< gg "K" >}} からの暴れ潰し。開幕に狙える。
    目押しではなく超連打がよい。{{< gg "2D" >}} がカウンターになったらさらに追い討ちで {{< gg "d.K jc.HS j.D(x2) air-dash j.K j.D 張付" >}}
  - 崩し
    - {{< gg "d.K dc.投げ" >}}（Dash, K, Dash をタタタンと押したら、あとは左右どちらか＋D連打でOK）
    - {{< gg "d.K jc.66BRC JHS" >}} 低空ブルーロマキャン
    - {{< gg "d.5HS > 214HS(シャーヴィ) ~ 4S [6](めくりポエンチ)" >}}。入力難度高。
    - {{< gg "d.K jc.236236HS(テンペ)" >}} 2.00以降はガードされたときの隙が大きいので危険
  - 置き
    - ジャンプしない相手には単純に {{< gg "2D" >}} を置いておく。

- 空中ダッシュからのコンボ
  - {{< gg "air-dash j.K j.D land f.S(x3) HS > 214K(セパル)" >}}。{{< gg "j.K" >}} は相手の頭の高さくらいで出すこと。ジャンプ攻撃の2段目を {{< gg "j.D" >}} にすることを間違えなければ大体繋がる。垂直ジャンプからのダッシュなど、相手に近いときは 1 段目は {{< gg "j.S" >}} の方がよいかも。

- 空中青ロマンキャンセルから急襲
  - {{< gg "jump 66BRC j.HS" >}} を高速に入力すると、空中ダッシュから急襲できる。ただの {{< gg "66BRC" >}} になってしまったら失敗。{{< gg "j.HS" >}} を連打しておくとよい。

- トロヴァォン
  - ガードされたら
    - {{< gg "2K 2D" >}} で暴れ崩し
  - ヒットしたら
    - {{< gg "2K > 214K(セパル)" >}}。2K の代わりに K でも可能（両方発生6Fでほぼ同じだが、2K の方がリーチが長いので 2K でよい）
    - 赤ロマキャン
      - {{< gg "88RRC hjc.HS j.D(x2) air-dash j.D(x2) HS 214K(セパル)" >}}。
        空中ダッシュ以降は連打気味で。
        ちょっと難しいけど画面中央から壁割までいける。
        でも次のスライドなし赤ロマキャンの方がダメージ高かったりする。
      - {{< gg "RRC 少し歩く c.S 2HS > 214S(ポエンチ) c.S jc.HS j.D(x2) air-dash j.K j.D" >}}。空中ダッシュからは連打気味で。

### 対空

- {{< gg "6P > 214K(セパル) dash K 6HS(x3)" >}}
- {{< gg "step 623S(ナセンテ)" >}}。当たれば {{< gg "dash K jc.HS" >}} での追い討ちが可能
- 遠目なら {{< gg "2D" >}} を置いておくのもあり
- 相手の着地に合わせて {{< gg "2S" >}} をおいておく。
- 近距離でのジャンプを読めれば {{< gg "空中投げ" >}} も頑張る。

### 開幕

- 様子見で後ろ歩き。しゃがみガード。→ {{< gg "d.2S" >}}
- {{< gg "HS counter-hit 214HS(シャーヴィ) 6S(ナセンテ) K 6HS(x3)" >}}
- {{< gg "d.K 2K 2D" >}}

### 投げからの連携

- （詐欺飛び・画面端）{{< gg "投げ（追い討ち）c.S jump-cancel air-dash j.D" >}}。
  まずは、{{< gg "c.S" >}} 以降の連携だけから練習。自分が起き上がった瞬間に {{< gg "c.S" >}} を押すとちょうどよい。
- （詐欺飛び・中央）{{< gg "投げ j.D" >}}。投げ後に前ジャンプ入れっぱ→ギリギリで {{< gg "j.D" >}} するだけ。
- （画面端）{{< gg "投げ (追い討ち)c.S HS(空振り) c.S" >}}。
  - 相手の投げ間合いの外から {{< gg "c.S" >}} を重ねられる。
- （中央）{{< gg "投げ 214HS(シャーヴィ) ~ 4S(ポエンチ)" >}}。ポエンチ中に {{< gg "[6]" >}} でめくり、{{< gg "[4]" >}} で表になる。
