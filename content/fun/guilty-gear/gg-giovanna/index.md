Guilty Gear Strive のジオヴァーナ練習メモです。
だんだん長くなってきたので [GGST基礎練メニュー](/p/73d9ami/) から分離しました。
ジオは他のキャラと比べてずるさを感じにくいのでよいです。
逆にいうと簡単にガードを崩せるような連携がないので、いろいろ工夫して動く必要があります。

コマンド
----

- セパルトゥア {{< gg "214K(セパル)" >}}。連携／コンボの締めに使う。ガードされたあとは確反にはならないが、相手ターンになってしまうので基本ガードで耐える。
- トロヴァォン {{< gg "236K(トロヴァ)" >}}。突進技。ガードさせれば有利なので固め継続できる。慣れている人には {{< gg "6P" >}} で簡単に落とされる。
- ソウ・ナセンテ {{< gg "623S(ナセンテ)" >}}。無敵時間のない対空技。ステップから出すのも有効。
  {{< gg "623S(近距離ナセンテ) BRC" >}} のガード崩しも強力。
  ガード時はジオ側 -4F なので、ガード、バクステ、{{< gg "2K" >}}、{{< gg "6P" >}}、{{< gg "623S(ナセンテ)" >}} などでフォロー。
- ソウ・ポエンチ {{< gg "236S(ポエンチ)" >}}。飛び上がり攻撃。
  {{< gg "2HS" >}} からのコンボに組み込むのが基本。
  空中では前後に移動可能。
  ガードされた場合は {{< gg "(空中)RRC j.P(x2)" >}} でガード崩し。
  {{< gg "めくりj.K > 236S(空中ポエンチ)" >}} というコンボもあり。
- シャーヴィ: {{< gg "214HS(シャーヴィ)" >}}。
  - 派生 {{< gg "6K" >}} トロヴァ。ガードさせて有利。ヒットしたらコンボへ。無敵技のない相手には起き上がりにうまく重ねるとよい。
  - 派生 {{< gg "6S" >}} ナセンテ。相手が浮いている場合、ここからコンボの起点に（開幕 {{< gg "HS counter-hit" >}} 時など）。
  - 派生 {{< gg "4K" >}} セパル。コンボの締めに。中距離から急に出す手もあり。
  - 派生 {{< gg "4S" >}} ポエンチ。めくりに使える。{{< gg "dash HS > 214HS(シャーヴィ) 4S(ポエンチ)" >}} でめくりからのコンボも可能。
- ヴェンターニア（覚醒）{{< gg "632146HS(ヴェンタ)" >}}。唯一のリバサ無敵技。ガードされたら終了。
- テンペスターヂ（覚醒）{{< gg "236236HS(テンペ)" >}}。2.00 以降はガードされると不利になったため使い所が減った。


立ち回り／ガード崩し／セットプレイ
----

### 対空

- {{< gg "6P > 214K(セパル) dash K 6HS(x3)" >}}
- {{< gg "dash 623S(ナセンテ) dash K jc.HS ..." >}}
- 相手の着地に合わせて {{< gg "2D" >}} や {{< gg "2S" >}} をおいておく。
- 画面端で相手が飛んだら気合いで {{< gg "空中投げ" >}}。

### 起き攻め／近距離

ガードされたときに {{< gg "214K(セパル)" >}} まで繋げても全部ガードされるだけなので、いかに「投げ」や「ディレイ」、「最速ダストロマキャン」、「低空ロマキャン」、「めくりポエンチ」で揺さぶっていくかがポイント。

- {{< gg "c.S dash-cancel 投げ" >}}。ガード崩し。
- {{< gg "c.S dash-cancel c.S" >}}。固め継続。
- {{< gg "c.S delay 2S delay HS" >}} バクステ狩り＆暴れ潰し。ヒット時は {{< gg "214K(セパル)" >}}、ガード時は {{< gg "236K(トロヴァ)" >}} 固め。
- {{< gg "c.S delay f.S(x3) delay HS" >}} 暴れ潰し。
- {{< gg "c.S f.S(x3) HS > 214HS(シャーヴィ) ~ 4S(ポエンチ) [6] 2K 6HS(x3)" >}}。連携をガードされたときのポエンチめくり崩し。ポエンチが当たったら、{{< gg "2K" >}} で拾える。
- 中段
  - {{< gg "c.S 5D 66RCC" >}}。近Sからの最速ダスト中段（直でもOK）。テンションゲージ溜まっているとき限定。
- めくり／表裏
  - {{< gg "214S(ポエンチ) ~ [6](めくり)" >}} / {{< gg "214HS(シャーヴィ) ~ 4S(ポエンチ) ~ [6](めくり)" >}}。
    - いずれも {{< gg "6" >}} 入れっぱでめくりになる。シャーヴィからの方がダメージ大。
- ★ジャンプを絡めた崩し
  - 通常ジャンプ＋ダッシュ混ぜ。頂点あたりで {{< gg "j.HS" >}} を当てる。
    - {{< gg "j.HS air-dash j.HS j.D" >}}
    - {{< gg "j.HS j.D air-dash j.HS" >}}
  - 低空青ロマ／F式
    - {{< gg "c.S jamp-cancel 66BRC j.HS (land) j.HS" >}}。低空青ロマ中段 → F式。
    - {{< gg "c.S jamp-cancel j.HS 22RRC land 2S HS" >}}。下スライド赤ロマ下段。
- {{< gg "シミー → 空投げ" >}} ファジージャンプ返し＆リバサ昇竜すかし。相手が飛ぶかだけに集中して **空投げ** 狙い。
- {{< gg "c.S f.S(x3) HS > 236K(トロヴァ) 2K 2D 236K(トロヴァ) ..." >}} なめぷ永久トロヴァ固め。{{< gg "6P" >}} に負ける。画面端でトロヴァがヒットしたら追撃可能。

### 中距離

- 地上ダッシュからの連携
  - {{< gg "dash 2D > 214K(セパル) 2K > 214K(セパル)" >}}。
  - {{< gg "dash K 6P > 214K(セパル)" >}}（画面端追撃）{{< gg "K > 214K(セパル)" >}}（中央追撃はムリ）
  - {{< gg "dash K 2D > 214K(セパル)" >}}（画面端追撃）{{< gg "K > 214K(セパル)" >}}（中央追撃）{{< gg "2K > 214K(セパル)" >}}
  - セパルがカウンターヒットしたら {{< gg "dash c.S" >}} からのポエンチコンボや {{< gg "d.K" >}} からの空中コンボへ。
  - 崩し
    - {{< gg "dash K dc.投げ" >}}（Dash, K, Dash をタタタンと押したら、あとは左右どちらか＋D連打でOK）
    - {{< gg "dash K jc.66BRC JHS" >}} 低空ブルーロマキャン
    - {{< gg "dash 5HS > 214HS(シャーヴィ) ~ 4S [6](めくりポエンチ)" >}}。入力難度高。
    - {{< gg "dash K jc.236236HS(テンペ)" >}} 2.00以降はガードされたときの隙が大きいので危険

- 空中ダッシュからのコンボ
  - {{< gg "air-dash j.K j.D land f.S(x3) HS > 214K(セパル)" >}}。
    空中ダッシュからの空中で 2 段ヒットさせるには {{< gg "j.K j.D" >}} の組み合わせくらいしかない。
    {{< gg "j.K" >}} は高め（相手の頭くらい）で出すこと。
    垂直ジャンプからのダッシュなど、相手に近いときは 1 段目は {{< gg "j.S" >}} の方がよいかも。

- バックジャンプからの急襲
  - {{< gg "back-jump 66BRC j.HS" >}} をタタンと入力すると、空中ダッシュから急襲できる。
    ただの {{< gg "66BRC" >}} になってしまったら失敗。
    {{< gg "66BRC" >}} の発動直後に {{< gg "j.HS" >}} を単独入力するとよい。

- トロヴァォン
  - ガードされたら ... {{< gg "2K 2D delay 214K(セパル)" >}} で暴れ崩し。セパル後は不利なので基本ガード。
  - ヒットしたら ... {{< gg "2K > 214K(セパル)" >}}。2K の代わりに K でも可能（両方発生6Fでほぼ同じだが、2K の方がリーチが長いので 2K でよい）
  - 赤ロマキャン
    - {{< gg "88RRC hjc.HS j.D(x2) air-dash j.D(x2) HS 214K(セパル)" >}}。
      空中ダッシュ以降は連打気味で。
      ちょっと難しいけど画面中央から壁割までいける。
      でも次のスライドなし赤ロマキャンの方がダメージ高かったりする。
    - {{< gg "RRC 少し歩く c.S 2HS > 214S(ポエンチ) c.S jc.HS j.D(x2) air-dash j.K j.D" >}}。空中ダッシュからは連打気味で。

### 開幕

- 様子見で後ろ歩き。しゃがみガード。→ {{< gg "d.2S" >}}
- {{< gg "HS counter-hit > 214HS(シャーヴィ) ~ 6S(ナセンテ) 2K 6HS(x3)" >}}
- {{< gg "dash K 2K 2D" >}}
- {{< gg "214S(ポエンチ)" >}}

### 投げからのセットプレイ

- （詐欺飛び・画面端）{{< gg "投げ（追い討ち）c.S jump-cancel air-dash j.D" >}}。
  まずは、{{< gg "c.S" >}} 以降の連携だけから練習。自分が起き上がった瞬間に {{< gg "c.S" >}} を押すとちょうどよい。
- （詐欺飛び・中央）{{< gg "投げ j.D" >}}。投げ後に前ジャンプ入れっぱ→ギリギリで {{< gg "j.D" >}} するだけ。
- （画面端）{{< gg "投げ (追い討ち)c.S HS(空振り) c.S" >}}。
  - 相手の投げ間合いの外から {{< gg "c.S" >}} を重ねられる。
- （中央）{{< gg "投げ 214HS(シャーヴィ) ~ 4S(ポエンチ)" >}}。ポエンチ中に {{< gg "[6]" >}} でめくり、{{< gg "[4]" >}} で表になる。


コンボ
----

### ポイント

立ち回りから {{< gg "dash f.S" >}} or {{< gg "dash 2S" >}} がヒットしたらシャビナセから {{< gg "2K" >}} で拾うルートへ。
{{< gg "2K" >}} は連打してるとスカることがあるので、目押しで単発で押すこと（ナセンテの着地くらいのタイミングで先行入力気味に押すだけでOK）。
画面中央から端まで運べる。

- {{< gg "2S(2) 2HS > 214HS(シャーヴィ) ~ 6S(ナセンテ) 2K 6HS(x3) (land) c.S(追い討ち)" >}}

{{< gg "dash c.S" >}} がヒットしたら下記のようなポエンチ繋ぎでフルコン大ダメージ。
**特に {{< gg "f.S" >}} を 1〜2 段目まで入れる場合は「ポエンチ」、3 段目まで入れる場合は「シャーヴィ派生ポエンチ（シャビポエ）」にしないとスカるので注意**。
要するに、距離がちょっと離れた場合はシャビポエにする必要がある。
シャビポエにすると相手との位置が入れ替わるので、これを利用して左右の壁に近い方に運ぶようにする。

- {{< gg "c.S f.S(2) 2HS > 214S(ポエンチ)" >}}
- {{< gg "c.S 2S(2) 2HS > 214S(ポエンチ)" >}}
- {{< gg "c.S f.S(3) 2HS > 214HS(シャーヴィ) ~ 4S(ポエンチ)" >}} （位置入替）

画面中央から始動したときは、ポエンチ後は空中コンボにすると比較的安定して壁割りまでいける。
下手に中央で拾い続けるより威力大。

- （ポエンチ後）{{< gg "c.S jc.HS j.D(2) air-dash j.K j.D" >}}（最後は {{< gg "j.HS j.D" >}} でも可）。

画面端までギリギリで運んだときは、着地後最速の {{< gg "(dash) K > 214K(セパル)" >}} で拾う。
張り付いたらヴェンターニアか溜めダストで壁割フィニッシュ。

### 具体例

- 画面端の近S始動
  - {{< gg "dash c.S f.S(3) 2HS > 214HS(シャーヴィ) ~ 4S(ポエンチ) c.S > 214HS(シャーヴィ) ~ 6S(ナセンテ) c.S HS (張付)" >}}
    - 最後の張り付き時は {{< gg "214HS(シャーヴィ)" >}} まで入れ込んでおけばよい。
      ゲージが溜まっていればヴェンターニアで壁割り。ゲージ回収のためにポエンチでフィニッシュでもよい。
  - {{< gg "dash c.S f.S(3) 2HS > 214HS(シャーヴィ) ~ 4S(ポエンチ) c.S jc.HS j.D(2) jc.HS (張付) 214S(空中ポエンチ)" >}}

- シャーヴィ派生トロヴァ始動
  - {{< gg "214HS(シャーヴィ) ~ 6K(トロヴァ) c.S f.S(x2) 2HS > 214S(ポエンチ) c.S f.S(x2) HS > 214K(セパル)" >}}
    - 最初の {{< gg "f.S" >}} は 2 段目で止めないと次の {{< gg "2HS" >}} が届かない。

- 溜めダスト始動
  - {{< gg "HS S jc S HS S jc HS HS" >}}
    - {{< gg "S" >}} と {{< gg "HS" >}} は大体入れ替え可能だけどこの順番が安定。
    - 空中ジャンプは確実に前ジャンプになるように、気持ち {{< gg "6 8" >}} の順番で入力すること。

- 溜めなしダスト（赤ロマキャン）始動
  - {{< gg "5[D] 66RCC c.S 2HS c.S 2HS > 214S(ポエンチ) c.S f.S(x3) 214K(セパル)" >}}
  - {{< gg "5[D] 66RCC dash f.S(3) 2HS > 214HS(シャーヴィ) ~ 4S(ポエンチ) c.S 2HS c.S f.S(x3) HS > WA ..." >}}

- 投げ → 赤ロマキャンから
  - （どこからでも）{{< gg "投げ 66RRC 溜めD ..." >}}
  - （中央）{{< gg "投げ 88RRC walk※  c.S 2HS > 214S(ポエンチ) c.S f.S(x3) HS > 214K(セパル) 張付" >}}
    - ※ダッシュで近づくとくぐってしまうので歩いて近づく
  - （端背負い）{{< gg "投げ RRC ダッシュでくぐる c.S 2HS > 214S(ポエンチ) c.S f.S(x3) HS > 214K(セパル) 張付" >}} 
  - （画面端）{{< gg "投げ RRC c.S 2HS > 214S(ポエンチ) c.S D 張付" >}}
  - （おまけ）魅せコンボ
    - （画面端）{{< gg "投げ RRC delay hj j.S j.HS j.D(x2) jdc.j.K j.D 張付" >}}

- （おまけ）テンション100%からの削り
  - {{< gg "d.K 6HS(x3) RRC j.HS j.D c.S 6HS(x3) RRC" >}}
